6년차 풀스택 개발자입니다. 화면을 구현하는 데서 멈추지 않고, 데이터로 가설을 검증하며 제품을 성장시키는 일에 집중합니다.

스튜디오 헤이너츠(개발 총괄)와 Plang(프론트엔드 리드)을 거쳐, 써밋플레이에서 리드 개발자로 합류해 CTO까지 맡았습니다. React Native·React·TypeScript로 앱과 웹을 만들고 Firebase 기반 백엔드·배포 자동화까지, 기획부터 운영까지 서비스의 전 과정을 다뤘습니다.

가장 중요하게 생각하는 것은 데이터 기반 의사결정입니다. Mixpanel을 직접 도입해 사용자 플로우를 분석하고 가설을 빠르게 검증해 왔습니다. 일례로 리믹스 콘테스트 웹을 데이터 분석에 근거해 리뉴얼하여, 첫 화면 이탈률을 52%에서 절반으로 낮추고 한 달 만에 MAU를 2배로 늘렸습니다.

최근에는 AI를 개발 프로세스에 통합하는 데 집중합니다. 멀티 에이전트 개발 파이프라인·배포 자동화를 구축하고, 영상의 음성을 추출·요약해 학습 콘텐츠를 자동 생성하는 서비스 등 반복을 자동화하는 도구를 만들어 왔습니다.

복잡한 문제를 단순한 구조로 정리하고 팀 전체의 생산성을 높이는 개발자, 기술이 실제 비즈니스 가치로 이어지게 만드는 개발자가 되고자 합니다.
